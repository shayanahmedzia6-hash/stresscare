## 💻 How to Use

### 1. Open in Browser
```
1. Navigate to the project folder
2. Open index.html in your browser
3. Or use a live server (VS Code Live Server extension recommended)
```

### 2. Explore Screens
```
- Click any screen link from the grid
- OR click "Launch App" to start from splash screen
- Use back buttons or bottom navigation to move between screens
```

### 3. Test Responsiveness
```
- Open Browser DevTools (F12)
- Toggle device toolbar (Ctrl+Shift+M / Cmd+Shift+M)
- Resize to test different breakpoints
```

### 4. Customize Content
```
- Replace placeholder text with real content
- Update colors in css/common.css
- Add your own images in /images/ folder
- Connect forms to backend API in js/main.js
```
